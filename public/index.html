Uncertainty Calculation
ตัวอย่าง: STD 0.02 bar → u_standard ≈ 0.02/√3
ตัวอย่าง: res 0.001 bar → 0.001/(2√3)
อิง dev ที่จุด 0 bar
SD ของ deviation ทุกจุด / √n
ค่าเฉลี่ยฮิสเทอรีซีส
วิธีคำนวณความไม่แน่นอน (คำอธิบายย่อ)
1) u_standard = Accuracy_STD / √3  |  2) u_res,std = Resolution_STD / (2√3)  |  3) u_zero = max(|dev@0|) / √3  |  4) u_repeat = SD(deviation) / √n  |  5) u_hyst = mean(|hysteresis|) / √3
U (k=2) = 2 × √( u_standard² + u_res,std² + u_zero² + u_repeat² + u_hyst² )

Formula & Guide (ละเอียด)

สูตรหลักสำหรับการสอบเทียบ
  1. แก้ความดันตามระดับความสูง: ΔP = (ρf − ρa)·g·Δh
  2. Zero correction: N₁,ⱼ = U₁,ⱼ; N₂,ⱼ = (S₁,ⱼ − S₂,ⱼ)+U₂,ⱼ; M₁,ⱼ = N₁,ⱼ − N₁,₀; M₂,ⱼ = N₂,ⱼ − N₁,₀
  3. แปลงสัญญาณเป็นความดัน (4–20mA): P = Pmin + (I−4)/(16)·FS
  4. ความไม่แน่นอนรวม: U = 2·√Σuᵢ² (k=2)
องค์ประกอบความไม่แน่นอน (ตัวอย่าง)
  • ustd = Accuracy/√3
  • ures,std = Resolution/(2√3)
  • uzero, urepeat, uhys
ขั้นตอนใช้งาน
  1. Setup → เลือกช่วงวัดและความแม่นยำ
  2. Entry → ใส่ STD/UUC ตามรอบ
  3. Uncertainty → ใส่ uᵢ หรือกด Auto-fill
  4. Report/Charts → ตรวจผล สร้าง PDF/CSV/Drive
